# chc-scripts
Scripts for installing and updating Chaincoin easily.

On a brand new VPS type:

sudo apt-get update

sudo apt-get install -y git 

git clone https://github.com/chaoabunga/chc-scripts

cd chc-scripts

chmod 777 install-chc.sh

./install-chc.sh

-----------
For update
-----------

chmod 777 install-chc.sh

./update-chc.sh
